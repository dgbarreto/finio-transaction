# finio-transactions

Módulo KMP de gerenciamento de transações financeiras do ecossistema Finio.

## Funcionalidades
- CRUD de transações
- Categorização e tags
- Sincronização offline-first
- Filtros por data e categoria

## Instalação

```kotlin
implementation("dev.finio:finio-transactions:VERSION")
```

## Dependências
- finio-auth

## Requisitos
- Android API 24+
- iOS 16+
- Kotlin 2.x
